-> Sistema de Seleção de Produtos em Java

--Projeto simples desenvolvido em Java utilizando `JOptionPane`, inspirado no funcionamento de máquinas de venda automática (vending machines).

-> Funcionalidades
- Exibição de menu de produtos
- Entrada de dados pelo usuário
- Validação de opções
- Tratamento de erros
- Seleção de produtos via interface gráfica simples

-> Tecnologias
- Java
- Swing (`JOptionPane`)
  
-> Objetivo
Projeto criado para praticar:
- Arrays
- Estruturas condicionais
- Tratamento de exceções
- Interfaces gráficas básicas com Swing

-> Inspiração
O sistema foi desenvolvido com base na lógica de máquinas automáticas de bebidas e snacks, onde o usuário escolhe um item através de um menu numérico.
