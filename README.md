-> Sistema de Seleção de Produtos em Java

-- Projeto simples desenvolvido em Java utilizando JOptionPane, inspirado em máquinas de venda automática (vending machines).

-> Objetivo
Projeto criado com foco em aprendizado de lógica de programação e manipulação de vetores (arrays) em Java.

-> Conceitos praticados
- Vetores (Arrays)
- Entrada de dados
- Conversão de valores
- Interface gráfica com JOptionPane
- Estrutura básica de aplicações Java

-> Tecnologias utilizadas
- Java
- Swing (JOptionPane)

-> Como executar
1. Compile o arquivo:
```bash
javac MaquinaSeletora.java
